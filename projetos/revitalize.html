Site Institucional Revitalize

Esse projeto surgiu a partir de um projeto de extensão da UFU com o objetivo de ajudar uma casa de acolhimento de idosos a divulgar seu trabalho.

Fui responsável pelo desenvolvimento do site com HTML, CSS e publicação via GitHub Pages, aplicando responsividade e personalização visual, aprendendo enquanto desenvolvia uma solução real.
