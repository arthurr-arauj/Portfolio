Site Institucional Revitalize

Um site responsivo para uma casa de acolhimento de idosos, desenvolvido como projeto de extensão na UFU.

Descrição do Projeto

Esse projeto surgiu a partir de uma iniciativa de extensão da UFU com o objetivo de ajudar uma casa de acolhimento de idosos a divulgar seu trabalho. Fui responsável pelo desenvolvimento completo do site com HTML e CSS, aplicando conceitos de responsividade e personalização visual para criar uma solução real e funcional.
